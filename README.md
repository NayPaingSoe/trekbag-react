### Prerequisites

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/trekbag_react.git
   cd trekbag_react
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173`
